# 🐾 Pet Booking App

A Flutter-based mobile application that allows users to explore and book various pet services like grooming, walking, cleaning, and boarding.

---

## 🚀 Features

* 🐶 View available pet services
* 📅 Book services easily
* 💰 Wallet balance system
* 📜 Booking history tracking
* ⚡ Smooth UI with loading shimmer effect
* 🔄 API integration with error handling

---

## 🛠️ Tech Stack

* **Flutter** (UI Framework)
* **Dart** (Programming Language)
* **Provider** (State Management)
* **REST API** (Data Fetching)
* **Material UI**

---

## 📱 Screens

* Home Screen (Services list)
* Service Detail Screen
* Booking Screen
* Confirmation Screen
* History Screen

---

## 📡 API Used

* https://jsonplaceholder.typicode.com/posts

---

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/YogeshDadheechi/Pet_Booking_App.git
   ```

2. Navigate to project folder:

   ```bash
   cd Pet_Booking_App
   ```

3. Install dependencies:

   ```bash
   flutter pub get
   ```

4. Run the app:

   ```bash
   flutter run
   ```
